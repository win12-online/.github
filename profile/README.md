# Hey, this is Windows12 Online 👋

We are responsible for maintaining the [Win12](https://github.com/win12-online/win12) project and a series of dependent services related to it.

## Contact Us
### Email Contact
For cooperation requests, media inquiries, complaints and reports, you can contact the project owner at: `starry-source@outlook.com`

For intellectual property complaints, please send an email to the above address and CC `ipcomplaint@win12.cloud` at the same time.

For community building, content maintenance, consulting suggestions, and other usage issues, please contact `generalinquiry@win12.cloud`.

Please try to leave detailed contact information in your message so that we can get back to you.

**Note**: Emails ending with `win12.cloud` are handled by @tangyuan0821.
